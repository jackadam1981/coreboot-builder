# EFI Shell 调试命令参考
# 进入 EFI Shell 后可以执行以下命令检查状态

# 1. 检查网卡和网络驱动
devices
# 查看所有设备列表

drivers
# 查看已加载的驱动

connect -r
# 重新连接所有设备

# 2. 检查网络协议
ifconfig -l
# 列出网络接口

ping 8.8.8.8
# 测试网络连通性

# 3. 查找 iPXE EFI 文件
map -r
# 重新映射所有文件系统

fs0:
# 切换到第一个文件系统（通常是 EFI 分区）
ls
# 列出当前目录

# 4. 检查 CBFS 中的文件
# 如果有 cbfstool 可用的话
# cbfstool coreboot.rom print
# 查看 ROM 中集成的文件

# 5. 手动启动 iPXE
# 如果找到 ipxe.efi 文件
ipxe.efi
# 或
fs0:\ipxe.efi

# 6. 检查启动选项
bcfg boot dump
# 查看当前启动配置

# 7. 查看内存映射
memmap
# 显示内存映射信息

# 8. 检查 PCI 设备
pci -l
# 列出 PCI 设备

# 9. 查看系统信息
dmpstore
# 查看 EFI 变量

# 10. 网络设备特定检查
# 如果看到网络设备，尝试：
ifconfig -s eth0
# 查看 eth0 接口状态
